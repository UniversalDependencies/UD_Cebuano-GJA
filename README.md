# Summary

UD_Cebuano_GJA is a collection of annotated Cebuano sample sentences randomly taken from three different sources:  community-contributed samples from the website Tatoeba (https://tatoeba.org/en/), a Cebuano grammar book by Bunye & Yap (1971) and Tanangkinsing's reference grammar on Cebuano (2011). This project is currently work in progress.
